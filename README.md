# next-github-star

```
实现一个 github starred repo 管理工具:
1. 管理工具支持移动端, web
2. 分别加载授权用户的 starred repo list(https://github.com/lxthyme?tab=repositories) 以及当前用户的 repo list(https://github.com/lxthyme?tab=stars)
3. 列表要支持分页
4. 筛选功能: 复刻 github 官方的筛选功能
5. 增加打标功能: 针对每一个 repo 可以从下拉列表中选中当前已有的 tag, 也可以新建 tag
6. 收藏功能: 针对每一个 repo 可以收藏/取消收藏
7. 备注功能: 针对每一个 repo 可以查看/添加/编辑备注
8. 可以筛选收藏/未收藏, 已备注/未备注, 已打标 tag/未打标 tag
9. starred 支持 star/unstarr 等基本操作


避免重复造轮子:
1. nextjs, antd 已有组件如果符合需求要求优先使用
2. 如果 github 上有开源优秀库能够满足需求, 也可以考虑使用
```
